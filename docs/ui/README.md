# 背单词应用 Web UI

这是一个为背单词应用设计的现代化Web界面，基于莱特纳盒子算法提供智能学习体验。

## 文件结构

```
docs/ui/
├── index.html       # 主页面 - 功能导航和快速统计
├── learning.html    # 学习界面 - 交互式单词练习
├── statistics.html  # 统计页面 - 详细的学习数据分析
└── README.md        # 本说明文档
```

## 功能特性

### 主页面 (index.html)
- **现代化设计**: 采用玻璃拟态(Glassmorphism)设计风格
- **功能导航**: 快速访问学习、统计和导入功能
- **实时数据**: 显示待复习单词数、正确率等关键指标
- **响应式布局**: 适配各种屏幕尺寸
- **动画效果**: 流畅的过渡和交互动画

### 学习界面 (learning.html)
- **多种题型支持**: 
  - 配对题: 中英文单词匹配
  - 填空题: 给出中文和字母提示
  - 翻译题: 英译中，支持语义匹配
- **实时反馈**: 答题后立即显示正确性
- **进度跟踪**: 实时显示学习进度
- **结果统计**: 学习完成后显示详细的答题统计

### 统计页面 (statistics.html)
- **数据可视化**: 使用Chart.js展示学习数据
- **莱特纳盒子分布**: 饼图展示各盒子单词分布
- **学习趋势**: 折线图展示每日学习进度
- **成就系统**: 显示获得的学习成就徽章
- **数据导出**: 支持导出JSON格式的统计数据

## 设计亮点

### 视觉设计
- **渐变背景**: 紫色系渐变营造学习氛围
- **玻璃拟态**: 半透明效果增强视觉层次
- **色彩系统**: 统一的配色方案提升品牌一致性
- **字体选择**: Noto Sans SC 确保中文显示效果

### 交互体验
- **悬停效果**: 卡片悬停时的缩放和阴影变化
- **加载动画**: 页面切换时的加载提示
- **通知系统**: 友好的操作反馈
- **键盘支持**: Enter键快速提交答案

### 技术实现
- **Tailwind CSS**: 快速构建响应式界面
- **Chart.js**: 专业的数据可视化
- **原生JavaScript**: 无需额外框架依赖
- **模块化设计**: 清晰的代码结构

## 使用方法

1. **访问主页面**: 打开 `index.html` 进入应用主页
2. **开始学习**: 点击"立即开始"按钮进入学习界面
3. **查看统计**: 点击"查看统计"查看详细数据
4. **导入单词**: 点击"选择文件"导入TXT格式的单词文件

## 单词文件格式

支持的TXT文件格式：
- `中文 英文` (空格分隔)
- `中文\t英文` (Tab分隔)

示例：
```
苹果 apple
香蕉 banana
橙子 orange
```

## 语义匹配

翻译题型支持语义相似性匹配，例如：
- 正确答案: "点心"
- 用户答案: "零食" / "小食" / "小吃" → 判定为正确

## 后端集成

这些界面设计为与Node.js后端集成：

### 连接点
- **数据加载**: 从后端API获取单词库和学习数据
- **答案评估**: 调用后端的 `evaluateAnswer` 方法
- **状态保存**: 将学习进度保存到后端数据库
- **统计计算**: 实时计算和更新学习统计

### API接口建议
```javascript
// 获取学习数据
GET /api/words/review

// 提交答题结果
POST /api/quiz/submit

// 获取统计数据
GET /api/statistics

// 导入单词
POST /api/words/import
```
